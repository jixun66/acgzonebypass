# English

## What is this script does?
By pass the acgzone.us access restriction.

## General Requirements
* php environment
* Curl function support.
* MySQL Database (Not needed if you are try to running without cache system)
* An account or logged in cookie for acgzone.us

## User requirements
* A browser supports Greasemonkey script
* A Greasemonkey script which allowed to change it's server address at http://userscripts.org/scripts/show/149762

## Config script
* View the php file for help (However the annotation is in Chinese)

绕过 acgzone.us 需要登录的限制。
